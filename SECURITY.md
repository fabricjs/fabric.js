# Security Policy

## Reporting Security Issues

If you believe you have found a security vulnerability, we encourage you to let us know right away.

We will investigate all legitimate reports and do our best to quickly fix the problem.

We prefer you use the github specifi tool:
https://github.com/fabricjs/fabric.js/security/advisories/new

If you can't, please report any vulnerabilities in our open source repositories to andreabogazzi79@gmail.com.
