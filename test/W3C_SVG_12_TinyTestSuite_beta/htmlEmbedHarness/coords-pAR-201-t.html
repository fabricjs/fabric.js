This test verifies the implementation of the viewBox and the preserveAspectRatio attribute on the animation element. It exercises the various preserveAspectRatio values and uses a general entity definition in order to make reading of the SVG source easier.

For the test to pass the following criteria should be met:

  1. the xMin* test image left edge should be on the left edge of the blue viewport rect
  2. the xMid* test image should be centered in the blue viewport rect
  3. the xMax* test image right edge should be on the right edge of the blue viewport rect
  4. the yMin* test image top edge should be on the top edge of the blue viewport rect
  5. the yMid* test image should be centered in the blue viewport rect
  6. the yMax* test image bottom edge should be on the bottom edge of the blue viewport rect
  7. no part of the image may be clipped away
